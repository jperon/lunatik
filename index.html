Lunatik is a framework for scripting the Linux kernel with Lua.

Modules

completion Lua bindings for kernel completion mechanisms.
data Provides Lua bindings for direct memory access and manipulation.
fib Provides Lua bindings to the Linux kernel's Forwarding Information Base (FIB) rules.
fifo Provides Lua bindings to the Linux kernel's kfifo (kernel FIFO) implementation.
device Low-level Lua interface for creating Linux character device drivers.
notifier Provides Lua bindings to the Linux kernel's notifier chain mechanism.
probe Provides Lua bindings to the Linux kernel's kprobes mechanism.
syscall Provides Lua bindings for accessing kernel system call information.
lunatik Core Lunatik module.
lunatik.runner Manages the execution and lifecycle of Lunatik scripts.
linux Provides Lua bindings to various Linux kernel facilities.
netfilter Low-level Lua interface to the Linux Kernel Netfilter framework.
